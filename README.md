# RHD-Atlas-AI
 An AI-powered public health platform for early screening, risk assessment, and epidemiological visualization of Rheumatic Heart Disease across Africa.

 This Branch include the Word file, with the Full Project Documentation and Problem Statement...
 
## How to navigate?
 Each PArt of the Project is presented in each of the github repository's branches as follows:
  1. App Branch: Contains the main app files and instructions on How to run it...
  2. Data Branch: The raw, uncleaned data we gathered without any cleaning or EDA...
  3. Notebooks Branch: The cleaned Data plus the combined master notebook where we explored the Data, Audited it, Cleaned it, and made a Dictionary to structure the workflow...
